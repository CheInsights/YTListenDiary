# YTListenDiary
Personal deep-dive into my own YouTube Music watch history (exported via Google Takeout).

Turned raw JSON chaos into clean insights: who I loop the most, when I actually listen, and which hours/days I go hardest.

### 🔥 What you'll find inside

- 🎯 **Replay Energy Leaderboard** — most repeated songs & artists (the real obsession detector)
- 🕒 **Listening Heatmap** — hour-of-day × weekday patterns (Saturday 2 PM supremacy unlocked)
- 📈 Monthly trends & binge session hints
- Clean data pipeline: artist extraction, title cleanup, timezone fixes, Unknown filtering

### Key discoveries so far

- Saturday afternoons = peak binge zone (240+ plays in a single hour 😭)
- Weekday sweet spot: 4–7 PM unwind sessions
- Basically zero late-night plays — not built for 3 AM scrolling
- Trap/rap rotation heavy: Yeat, XXXTENTACION, Kendrick, BabyDaiz & friends
